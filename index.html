CRYSTALS COLLECTOR

How to play

The game starts with a random target number on the screen.

There are 4 crystals and each one has a different random value assigned.

The objective of the game is to click on the crystals so that their value adds up and matches the target number given

If your score counter matches the target number you will win but if you happen to have more points then you will lose.

The wins and losses will be recorded and the game will restart automatically after each round.
